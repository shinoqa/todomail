EmailJS 設定
EmailJS を使うと、バックエンド不要でメール送信できます。
1. emailjs.com でアカウント作成（無料）
2. Gmail / Yahoo のサービスを追加
3. メールテンプレートを作成（変数: to_email, subject, message）
4. Public Key / Service ID / Template ID を入力
